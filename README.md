# 🧩 CubeSolver Pro

A professional Rubik's Cube Simulator built with **React**, **Three.js**, and a custom cube engine.

---

## ✨ Features

- 🎲 Interactive 3D Rubik's Cube
- 🔄 R L U D F B Moves
- 🎬 Smooth Face Animations
- 📜 Move History
- 🎯 Scramble Generator
- ↩ Undo / Redo
- 🤖 Solver *(Coming Soon)*
- 📚 Learn Mode *(Coming Soon)*
- ⏱ Speedcubing Timer *(Coming Soon)*

---
